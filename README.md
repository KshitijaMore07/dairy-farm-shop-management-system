🥛 Dairy Farm Shop Management System

A web-based Dairy Farm Shop Management System developed using PHP and MySQL. This system helps manage dairy products, categories, orders, and admin operations efficiently.

📌 Project Overview

The Dairy Farm Shop Management System is designed to automate and manage dairy shop activities such as:

Product Management

Category Management

Order Management

Admin Login System

Dashboard Overview

This project is useful for small dairy businesses to manage their shop digitally.

🛠️ Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Server: XAMPP (Apache & MySQL)

📂 Project Structure

dfsms/
│── admin/
│── includes/
│── css/
│── js/
│── index.php
│── config.php

admin/ → Admin panel files

includes/ → Database connection

css/ → Styling files

js/ → JavaScript files

index.php → Main login page

⚙️ Installation Steps

Install XAMPP.

Copy the project folder inside:
C:\xampp\htdocs\
(Or your XAMPP installation folder htdocs)

Start Apache and MySQL from XAMPP.

Open phpMyAdmin.

Create a new database named: dairyshop

Import the provided .sql file into dairyshop database.

Open browser and type:
http://localhost/dfsms
